## Setup Environment
* Install NodeJS LTS(Latest Stable Version) from https://nodejs.org/en/download/.

## Run API server
* From root `cd api`
* Run `npm i`
* Start server with `npm run dev`

## Run Frontend
* From root `cd web`
* Run `npm i`
* Start server with `npm start`
